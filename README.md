# Habit Tracker

A simple weekly habit tracker built with HTML, CSS, and JavaScript.

The app lets users add daily habits, mark progress on a weekly grid, move between weeks, and keep their data saved after refreshing the page.


## Run locally

No packages are required. From the project folder, run:

python3 -m http.server 5173


Then open:

http://localhost:5173


If `python3` is not available on Windows, try:

python -m http.server 5173


You can also open `index.html` directly in a browser, but the local server command is the cleaner option.
